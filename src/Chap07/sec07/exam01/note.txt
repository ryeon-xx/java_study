클래스 형변환(다형성)
변수에 들어있는 주소를 다른 변수에 넣을 때
타입을 변환해서 할당


크기 기준으로 (큰지 작은지만 판단하면 개념은 명확해짐)
자동형변환
- 작은 곳에 있던 데이터가 큰 곳으로 복사되는 경우, 자동으로 형변환되어 저장

강제형변환
- 큰 곳에 있던 데이터가 작은 곳으로 복사되는 경우, 강제로 형변환하여 저장


자동 타입 변환
- 자동적으로 타입 변환이 일어나는 것
부모타입 변수 = 자식타입객체;

- 자식은 부모의 특지과 기능을 상속받기 때문에 부모와 동일하게 취급

참조형 형변환
: 클래스는 상속관계에 있을 때만 가능
    -> 상속만 대소관계가 생김.


자동형변환인 경우 자식의 메소드는 호출 불가능!
