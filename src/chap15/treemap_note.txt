/*
 * ===== TreeMap =====
 *
 * - 이진 트리를 기반으로 한 Map 컬렉션
 * - 키:값 쌍으로 저장, 키를 기준으로 자동 정렬, 검색에 특화 (속도 매우 빠름)
 * - 부모 키 값과 비교해서
 *   낮은 것 → 왼쪽 자식 노드
 *   높은 것 → 오른쪽 자식 노드에 Entry 객체 저장
 *
 * TreeMap<K, V> treeMap = new TreeMap<K, V>();
 * TreeMap<K, V> treeMap = new TreeMap<>();
 *
 *
 * [ 주요 메서드 ]
 *
 * firstEntry()                      - 제일 낮은 Map.Entry 리턴
 * lastEntry()                       - 제일 높은 Map.Entry 리턴
 * lowerEntry(K key)                 - 주어진 키보다 바로 아래 Map.Entry 리턴
 * higherEntry(K key)                - 주어진 키보다 바로 위 Map.Entry 리턴
 * floorEntry(K key)                 - 동등한 키가 있으면 해당 Map.Entry 리턴,
 *                                     없으면 바로 아래 Map.Entry 리턴
 * ceilingEntry(K key)               - 동등한 키가 있으면 해당 Map.Entry 리턴,
 *                                     없으면 바로 위 Map.Entry 리턴
 * pollFirstEntry()                  - 제일 낮은 Map.Entry를 꺼내고 컬렉션에서 제거
 * pollLastEntry()                   - 제일 높은 Map.Entry를 꺼내고 컬렉션에서 제거
 * descendingKeySet()                - 내림차순으로 정렬된 키의 NavigableSet 리턴
 * descendingMap()                   - 내림차순으로 정렬된 Map.Entry의 NavigableMap 리턴
 *
 *
 * [ 범위 검색 메서드 ]
 *
 * headMap(K toKey, boolean inclusive)
 *   - 주어진 키보다 낮은 Map.Entry들을 NavigableMap으로 리턴
 *   - inclusive: true면 주어진 키 포함, false면 미포함
 *
 * tailMap(K fromKey, boolean inclusive)
 *   - 주어진 키보다 높은 Map.Entry들을 NavigableMap으로 리턴
 *   - inclusive: true면 주어진 키 포함, false면 미포함
 *
 * subMap(K fromKey, boolean fromInclusive, K toKey, boolean toInclusive)
 *   - 시작과 끝으로 주어진 키 사이의 Map.Entry들을 NavigableMap으로 리턴
 *   - fromInclusive: 시작 키 포함 여부
 *   - toInclusive: 끝 키 포함 여부
 */
